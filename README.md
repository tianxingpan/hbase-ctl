# hbase-ctl

## Install and Usage
Install the tool with:

```bash
go get -u github.com/tianxingpan/hbase-ctl
```

## upgrade
Upgrade the tool with:

```
hbase-ctl update
```

## Example

```shell
hbase-ctl get 'ns1:t1', 'r1'
```
